Project Name and Description:
	Soccer Vision

 Dynamic soccer manager and game simulator where you can manage your own soccer club. You can play minigames to boost your players' stats to improve your chances of winning matches against opposing teams. The minigames involve OpenCV and include heading a virtual ball and detecting juggling
    

How to run:
All the pictures and be attached. Save all the code in the same directory, and make folders accordingly based on the code and image zip file.

General instructions:
Click on any of the boxes to play. Instructions will be posted ingame. Press 'q' to close the tkinter window

Libary installations:
OpenCV and numpy are the main ones. I used module_manager to download these.
